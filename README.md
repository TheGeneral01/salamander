#Welcome to Salamander!

Salamander is a new progamming language built on customization, while also allowing you to experience the performance of pre-compiled C/C++, while having many features that Python implements. 
Syntax can be statically, or dynamically typed, however the rule of thumb is static -> dynamic, but not dynamic -> static.

The current build of the compiler is in a really rough state right now, but soon the parser should be up and running. Thanks for checking it out :D
