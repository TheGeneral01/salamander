# Welcome to Salamander!

Salamander is a progamming language built on customization, while also allowing you to experience the performance of pre-compiled C/C++, while having many features that Python implements. 
Syntax can be statically, or dynamically typed, however the rule of thumb is static -> dynamic, but not dynamic -> static.

Current status: The current build of the compiler is in a really rough state right now, but soon the parser should be up and running,

The reason that this compiles to C is so a more widely-supported C compiler can compile it to ASM, for now anyways. I will be working in the future to push directly from SAL -> ASM.
